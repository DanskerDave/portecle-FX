How to create a new keystore

To create a new keystore:

  1. From the File menu, choose New Keystore. Alternatively click on the New Keystore toolbar button:
  2. The New Keystore Type dialog is displayed. Select the desired keystore type using the radio buttons:
    • JKS: Java Keystore (Sun's Keystore format)
    • PKCS #12: Public-Key Cryptography Standards #12 Keystore (RSA's Personal Information Exchange Syntax Standard)
    • JCEKS: Java Cryptography Extension Keystore (More secure version of JKS)
    • BKS: Bouncy Castle Keystore (Bouncy Castle's version of JKS)
    • UBER: Bouncy Castle UBER Keystore (More secure version of BKS)
  3. Press the OK button.
  4. The title bar will change to display the text Untitled and the status bar will change to display the chosen keystore type.

Shortcut key: Ctrl-N.
