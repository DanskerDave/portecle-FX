How to clone a Keystore Key Pair entry

To clone (make an exact copy of) a Keystore Key Pair entry:

  1. Right-click on the Key Pair Keystore entry in the Keystore Entries table. Select the Clone item from the resulting pop-up menu.
  2. If you have yet to supply the password for the entry and the current Keystore type is not PKCS #12:
    • The Key Pair Entry Password dialog will be displayed.
    • Enter the Key Pair entry's password and press the OK button.
  3. The Cloned Key Pair Entry Alias dialog will be displayed.
  4. Enter the alias for the new key pair and press the OK button.
  5. If the current Keystore type is not PKCS #12:
    • The Cloned Key Pair Entry Password dialog will be displayed.
    • Enter the password to protect the key pair with, confirm it and press the OK button.
  6. The cloned key pair entry will appear in the Keystore Entries table.
