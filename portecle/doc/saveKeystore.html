How to save a KeyStore

To save a KeyStore to file:
  1. From the File menu, choose Save KeyStore. Alternatively click on the Save KeyStore toolbar button:
  2. If the KeyStore is not Untitled then it will be saved immediately. Otherwise...
  3. If you have yet to set a password for your Untitled KeyStore:
    • The Set KeyStore Password dialog is displayed.
    • Enter the password to protect the KeyStore with, confirm it and press the OK button.
  4. The Save KeyStore As dialog is displayed.
  5. Select the drive and folder where the KeyStore file is to be saved.
  6. Type the filename into the File Name text box.
  7. Click on the Save button.
Short-cut key: Ctrl-S.
