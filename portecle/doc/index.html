Portecle:

Portecle is a GUI version of the keytool command line program provided with the Java 2 SDK.

Portecle Help is not intended to be a guide to the use of keystores, certificates, and the like. What can be found here are "how to" guides describing how to carry out the specific keystore operations supported by Portecle. For more in-depth information on concepts such as X.509 certificates, public, private keys, etc. see the keytool documentation supplied with the Java 2 SDK.

Take the tour. Read the HOWTOs. Download: gz bz2 zip.
