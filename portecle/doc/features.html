Features

Portecle is a Java/ Swing implementation of the command-line keytool provided with the Java SDK. It can be used to graphically administer key pairs and the associated X.509 certificates of JKS, JCEKS, PKCS #12, BKS and UBER keystores. A complete list of Portecle's functionality is included below:
  • Create, load, save, and convert keystores.
  • Generate DSA and RSA key pair entires with self-signed Version 1 X.509 certificates.
  • Import X.509 certificate files as trusted certificates.
  • Import key pairs from PKCS #12 files.
  • Clone and change the password of key pair entries and keystores.
  • View the details of certificates contained within keystore entries and certificate files.
  • Export keystore entries in a variety of formats.
  • Generate certification requests (CSRs).
  • Import Certificate Authority (CA) replies.
  • Change the password of key pair entries and keystores.
  • Delete, clone, and rename keystore entries.
  • View the details of Certificate Revocation List (CRL) files.
