How to produce a KeyStore report

To produce a KeyStore report:
  1. From the Tools menu, choose KeyStore Report. Alternatively click on the KeyStore Report toolbar button:
  2. The KeyStore Report dialog will be displayed with details of the KeyStore in tree form including all entries and attached certificates.
  3. The report can be copied to clipboard in two formats:
    1. Press the Copy button to copy a plain-text report to the clipboard.
    2. Press the Copy as XML button to copy an XML report to the clipboard.
  4. After viewing the report acknowledge the dialog by pressing the OK button.
Shortcut key: Ctrl-R.
